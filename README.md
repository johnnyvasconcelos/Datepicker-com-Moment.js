# Custom Date Picker - Zoe Yacht

Componente customizado de seleção de datas desenvolvido para a plataforma de aluguel de cruzeiros da empresa turca Zoe Yacht. O sistema gerencia regras de negócio complexas de disponibilidade e fusos horários diretamente no frontend, e permanece funcional no site oficial.

## Tecnologias Utilizadas

* Bootstrap 5: Estruturação visual e responsividade.
* Moment.js: Manipulação, validação e lógica de datas.
* JavaScript: Lógica central do componente desenvolvida do zero.
